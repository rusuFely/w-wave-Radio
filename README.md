# W-Wave Radio

## 🔗 Live Demo
https://rusufely.github.io/w-wave-Radio/

## 📌 Descriere
Website pentru un radio online, realizat cu accent pe design modern, layout responsive și experiență vizuală atractivă.

## 🚀 Funcționalități
- structură multi-secțiune (hero, programe, informații)
- layout responsive pentru diferite dispozitive
- elemente vizuale atractive și organizare clară a conținutului
- navigare intuitivă

## 🛠️ Tehnologii utilizate
- HTML5
- CSS3 (Flexbox, Grid)

## 📱 Responsive
Da – optimizat pentru mobil, tabletă și desktop

## 🎯 Scop
Dezvoltarea abilităților de creare a interfețelor moderne și responsive, precum și organizarea eficientă a conținutului într-un website real.
